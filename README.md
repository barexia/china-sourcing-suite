# China Sourcing Guide

A comprehensive guide teaching English-speaking entrepreneurs how to source products from China for their e-commerce businesses.

## About

China Sourcing Guide is a premium PDF guide (80+ pages, 6 modules) that covers the entire sourcing process — from finding suppliers on 1688 to shipping products to your customers worldwide. No prior experience in China or Chinese language required.

### What's Inside

- **Module 1** — Why source from China (cost breakdown, platform comparison)
- **Module 2** — Getting started on 1688 (registration, navigation, search)
- **Module 3** — Finding & vetting suppliers (verification, samples, inspection)
- **Module 4** — Communication & negotiation (templates, payment, common scams)
- **Module 5** — Shipping & logistics (carriers, freight forwarders, customs)
- **Module 6** — Listing & selling (photography, pricing, Amazon/Shopify)
- **Appendices** — Phrasebook, templates, checklists, glossary

## Tech Stack

- **Framework:** Next.js 16 + React 19
- **Styling:** Tailwind CSS 3
- **Language:** TypeScript 5
- **Deployment:** Vercel
- **Payment:** Gumroad

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## PDF Generation

```bash
npm run generate-html   # Generate HTML from markdown
npm run generate-pdf    # Convert to PDF (requires Playwright)
```

## Deployment

Push to the main branch — Vercel auto-deploys.
